# ntfs

<img src="../../assets/fs/go-filesystems-ntfs.png" width="110" align="right">

Windows NT filesystem.

Implements [`filesystem.Filesystem`](../interface.md). No cgo, no root.

## Install

```bash
go get github.com/go-filesystems/ntfs
```
